# Amazon Frontend Clone (HTML & CSS)

A responsive frontend clone of Amazon built purely with HTML and CSS.  
This project replicates the look and feel of Amazon’s homepage, product sections, and navigation without using JavaScript — focusing entirely on layout, styling, and responsiveness.

## Description
The project demonstrates strong frontend skills, including:
- Clean HTML structure  
- Modern CSS design techniques (Flexbox, Grid)  
- Responsive design for desktop and mobile screens  

## Features
- Amazon-style homepage layout  
- Search bar design  
- Product listing cards  
- Navigation bar and menu  
- Fully responsive layout  
- Modern and clean design using only CSS  

## Technologies Used
- HTML5  
- CSS3
